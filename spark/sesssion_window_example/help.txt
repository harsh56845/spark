PYSPARK STRUCTURED STREAMING (SOCKET) - COMPLETE SETUP GUIDE (LINUX)

---

## STEP 1: INSTALL JAVA (REQUIRED)

# Install Java 17 (recommended for Spark 4.x)

sudo apt update
sudo apt install openjdk-17-jdk -y

# Verify installation

java -version

# Set JAVA_HOME

export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH

---

## STEP 2: INSTALL APACHE SPARK

# Download Spark

wget https://downloads.apache.org/spark/spark-4.1.1/spark-4.1.1-bin-hadoop3.tgz

# Extract

tar -xvzf spark-4.1.1-bin-hadoop3.tgz

# Move to /opt (optional)

sudo mv spark-4.1.1-bin-hadoop3 /opt/spark

# Set environment variables

export SPARK_HOME=/opt/spark
export PATH=$SPARK_HOME/bin:$PATH

# Verify

spark-submit --version

---

## STEP 3: INSTALL PYTHON + PYSPARK

# Install Python (if not installed)

sudo apt install python3 python3-pip -y

# Install PySpark

pip install pyspark

---

## STEP 4: CREATE SIMPLE CONSUMER (consumer.py)

from pyspark.sql import SparkSession

spark = SparkSession.builder 
.appName("SimpleSocketTest") 
.getOrCreate()

spark.sparkContext.setLogLevel("ERROR")

df = spark.readStream 
.format("socket") 
.option("host", "localhost") 
.option("port", 9999) 
.load()

query = df.writeStream 
.outputMode("append") 
.format("console") 
.start()

query.awaitTermination()

---

## STEP 5: START SOCKET SERVER

# Terminal 1

nc -lk 9999

---

## STEP 6: RUN SPARK CONSUMER

# Terminal 2

export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
spark-submit consumer.py

---

## STEP 7: SEND TEST DATA

# Terminal 3 (manual test)

echo "hello" | nc localhost 9999

## COMMON ERRORS + FIXES

1. ERROR: Java 25 / incompatibility
   Fix:
   → Always use Java 17

2. ERROR: Operation timed out
   Fix:
   → Start nc BEFORE Spark
   → Ensure data is being sent

3. ERROR: No output
   Fix:
   → Check producer or manual input
   → Reduce delays

4. ERROR: Port already in use
   Fix:
   → Kill process:
   lsof -i :9999
   kill -9 <PID>
